Reprsentation de donnes

Pour la rprsentation des donnes, nous allons voir trois mthodes diffrentes dans la suite : Isomap, LLE et MDS. Notez que ces trois mthodes permettent de rcuprer les coordonnes des points dans un sous-espace. Il est bien videmment possible (et mme recommander ventuellement) d'utiliser ces coordonnes pour raliser une opration telle que la classification (avec $k$-means ou une CAH).

Toutes ces mthodes sont incluses dans le sous-module manifold de la librairie sklearn.

Mthodes de classification

DBSCAN

Cet algorithme se base sur une notion de densit dans les donnes (dfinie globalement). Un individu est dit directement accessible un autre individu si la distance entre les deux est plus petite quun seuil dfini en amont. Et 2 individus sont dit accessibles entre eux sil existe une chane de points directement accessible 2 2 entre les deux. Une classe est donc compose dindividus accessibles entre eux.

Un point intressant est la capacit de la mthode dterminer que des points sont des outliers et donc ne les classer dans aucune classe.

In[9]:
from sklearn.cluster import DBSCAN

db = DBSCAN(eps = 0.75, min_samples = 10).fit(iris_num)
db

HDBSCAN

C'est une version hirarchique de DBSCAN, qui l'excute avec diffrentes valeurs d'eps et intgre le rsultat pour trouver un regroupement qui donne la meilleure stabilit. Cela permet de trouver des clusters de diffrentes densits (contrairement DBSCAN) et d'tre plus robuste la slection des paramtres.

hdb = HDBSCAN(min_cluster_size = 10).fit(iris_num)
